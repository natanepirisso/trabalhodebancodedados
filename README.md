Situação Banco de Dados

Site de Doações de pet

*Site , sem empresas, ongs ou qualquer outra coisa envolvida. apenas uma pessoa

*todos os pets foram achados pela detentora do site nas ruas

*Muitas pessoas podem adotar os pets e fazer doações de materiais que ajudem a detentora deste site a manter os pets++

+Pessoas *não* podem doar PETS+

+Materias podem ser desde Dinheiro até alimentos para os pets, brinquedos e objetos para conforto do pet+

*é necessario realizar uma inscrição no site para que a pessoa possa efetuar a adoção/doação

*adoções só podem ser realizadas presencialmente

*doações podem ser feitas via deposito, correios ou presencial

*o transporte de adoção e doação é feito de forma autonoma. logo, so pode ser efetuado caso a pessoa more perto(mesmo estado/cidade)

*a pessoa tambem pode ir buscar os pets e/ou levar os materias para a detentora do site, caso more perto.

*pessoas que moram longe so podem fazer doações.

*so é obrigatoria a adição do cartão de credito, caso a pessoa va usá-lo

*a pessoa pode agendar a sua ida ao local de adoção por meio de telefonemas,redes sociais ou whatsapp
